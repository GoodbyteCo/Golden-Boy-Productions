# Golden-Boy-Productions

## Operational Notes

### Local Development

Install dependencies:

```bash
npm install
```

| COMMAND       | DESCRIPTION                             |
|:--------------|:----------------------------------------|
| `npm run dev` | Run the site locally                    |
| `npm run img` | Convert all JPG images to Avif and WebP |
| `npm run fix` | Fix any linting errors                  |
